# 🎯 Checklist Goals

Aplicación **Full Stack** para gestionar objetivos personales.

El sistema permite a los usuarios crear metas, organizarlas y marcar su progreso hasta completarlas.
El objetivo del proyecto es practicar el desarrollo de una **aplicación completa con frontend, backend y base de datos**, utilizando tecnologías modernas y una arquitectura desacoplada basada en una API REST.

Este proyecto sirve como práctica para mejorar habilidades en **desarrollo web moderno, diseño de APIs y despliegue de aplicaciones en la nube**.

---

## Frontend

## Backend

* FastAPI
* Python

## Base de Datos

* PostgreSQL

---

# 🧩 Arquitectura

La aplicación sigue una arquitectura **Full Stack desacoplada**, donde el frontend consume datos a través de una API REST expuesta por el backend.

```text
Frontend (Vue)
      │
      │ HTTP / REST API
      ▼
Backend (FastAPI)
      │
      ▼
Database (PostgreSQL)
```

Este enfoque permite:

* Separación clara entre frontend y backend
* Escalabilidad del sistema
* Integración futura con otras aplicaciones o servicios
